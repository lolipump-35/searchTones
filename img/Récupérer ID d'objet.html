🔹 Solution avec un slider géré par une bibliothèque (ex: Splide.js)

Si tu utilises Splide.js, tu peux écouter l'événement de changement de slide et récupérer l'ID de la slide active :

javascript
var splide = new Splide('.splide').mount();

splide.on('active', function(slide) {
    let activeSlideId = slide.id; // Récupère l'ID de la slide active
    console.log("ID de la slide active :", activeSlideId);
});

👉 Ici, slide.id te donne l’ID de la slide active.

🔹 Solution sans bibliothèque (slider custom avec classes CSS)

Si ton slider est fait maison, avec des div contenant un class="active" pour la slide affichée, tu peux faire ceci :

javascript
function getActiveSlideId() {
    let activeSlide = document.querySelector(".slide.active"); // Sélectionne la slide active
    if (activeSlide) {
        console.log("ID de la slide active :", activeSlide.id);
        return activeSlide.id;
    }
    return null;
}

👉 Cette fonction récupère l'ID de la slide actuellement active.

Si ton slider change automatiquement, tu peux appeler cette fonction à chaque transition.

  • Si la méthode .includes(id) renvoie false (l'ID n'est pas encore dans le tableau), l'opérateur ! le transforme en true.
